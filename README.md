# InvenTrack 360 (project-development-log) 

This repository contains structured evidence of the development progress of the project.

  ## 📖 Overview
  InvenTrack 360 is an advanced robotic solution developed by Inbiodroid designed to automate inventory management and digitize logistics operations. By combining Autonomous Mobile Robots (AMR), Computer
  Vision, and Cloud Intelligence, it eliminates manual errors and provides real-time visibility into warehouse stocks.

  This project represents the first business case of the AnaDynamis ecosystem, leveraging telepresence and robotics to create inclusive operational models.

  ## 🚀 Key Modules & Development Areas

  This repository focuses on the following technical pillars:

  **Autonomous Navigation (AMR)**
   - Development of navigation stacks for indoor environments.
   - LIDAR-based mapping and obstacle avoidance.
   - Path planning for efficient warehouse traversal.

  **Computer Vision & Scanning**
   - Algorithms for detecting and decoding barcodes, QR codes, and pallet labels in motion.
   - Object recognition models to identify stock levels and placement issues.
   - Integration of high-resolution cameras for "Digital Twin" generation.


  **Cloud Platform & WMS Integration**
   - Backend logic for synchronizing robot data with the cloud.
   - API prototypes for communicating with Warehouse Management Systems (WMS).
   - Data visualization dashboards for real-time inventory tracking.


  **Teleoperation & AnaDynamis Core**
   - Remote control interfaces enabling human-in-the-loop supervision.
   - Low-latency video streaming and control protocols tailored for remote operators (including those with motor disabilities).
