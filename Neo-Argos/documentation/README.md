## 🚧 Work in Progress
Neo-Argos is currently a **Rapid Development & Research (R&D) platform**. As our team focuses on the real-world implementation of the ROS 2 stack, formal technical documentation (BOM, Schematics, API Guides) is still being compiled.

### Current Status
*   **Hardware:** Iterative prototyping phase (custom PCB testing).
*   **Software:** Tuning phase (Nav2 and EKF validation).
*   **Manuals:** Operating procedures are currently managed via internal team notes and SSH-based development workflows.

Formal documentation will be uploaded as the platform stabilizes and transitions from a prototype to a finalized service-robot foundation.
